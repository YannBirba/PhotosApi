# PhotosApi

## Laravel IDE Helper

### Générer les models helper

```bash
$ php artisan ide-helper:models
```
Dire oui lorsqu'il propose d'écrire la PhpDoc de la classe.

## Laravel Pint

```bash
$ composer pint
```

## Larastan

```bash
$ composer phpstan
```

## Pestle

```bash
$ composer test
```

## Check

```bash
$ composer check
```
